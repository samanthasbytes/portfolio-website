Meet App

The Meet App is a serverless progressive web application (PWA) built with React using a test-driven development (TDD) approach. It integrates with the Google Calendar API to display upcoming events and allows users to search for events by city, show or hide event details, specify how many events to display, use the app offline, add it to their home screen, and view charts showing event distribution and genre popularity. The goal of this project is to apply TDD, serverless architecture, and PWA principles to build a modern, responsive web application with high code quality, offline capability, and data visualization features.
